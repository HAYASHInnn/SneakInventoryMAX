## 使い方
**【注意事項】**
  
・マルチサーバー内にプレイヤーが１人であること 
 
・最大所持数が「64」のアイテムのみ適用 
 
・アイテム収集が不要になるため、達成感の喪失
 
**【使い方】**
 
１.pluginsフォルダにjarファイルを格納
 
２.Spigotサーバーの立ち上げる

３.Minecraftを起動後、マルチプレイを選択

４.ゲーム上でshiftキー（スニーク）を押す

## POINT
:+1: **ゲーム上の操作が簡単！！**

shiftキーのみで完結
